##学习总结
本周回顾练习了回溯法
###回溯是什么
回溯可以理解成梦境的归去来兮,有进入下一层梦境,有回到上个梦境,把结果带出来
###回溯解决什么问题
回溯可以解决,求所有结果的问题
###回溯模板
```
result = []int

backstrack(路径, 可选择列表):
    if 终止条件:
        result.add(路径)
        return
    for 选择项 in 可选择列表:
        路径.add(选择项)             //做选择
        backstrack(路径, 可选择列表) //进入下一个梦境
        路径.remove(选择项)         //撤销刚才的选择
```
